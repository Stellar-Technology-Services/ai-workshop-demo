# Resources

A curated dump of links to reach for after the workshop — when you want to go
deeper, copy a pattern, or set this up in your own stack. Grouped by tool/topic;
we'll keep adding to it.

---

- **[Agent customization overview](https://code.visualstudio.com/docs/agent-customization/overview)**
  — How to shape Copilot's agent in VS Code: custom instructions, reusable
  prompts, chat modes, and skills (the "folders and files" we author here).

- **[MCP in VS Code](https://code.visualstudio.com/api/extension-guides/ai/mcp)**
  — Connecting external tools and data to the agent via the Model Context
  Protocol.

- **[cursor.directory](https://cursor.directory/)**
  — Community directory of Cursor rules and MCP servers to browse and copy.

- **[cursorrules.org](https://cursorrules.org/)**
  — Collection of shareable `.cursorrules` files organized by stack and
  framework.

- **[agentskills.io](https://agentskills.io/home)**
  — The cross-harness Agent Skills standard — the same skill body, different
  wrapper across Copilot / Cursor / Claude Code.

- **[agentskills/agentskills](https://github.com/agentskills/agentskills)**
  — Open-source repo behind the standard: reference implementation and example
  skills.
